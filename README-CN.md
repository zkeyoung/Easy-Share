便捷、简单和易操作的在内网分享文件的玩具。基于Electron 和 TypeScript 构建。对应的前端项目为 [Easy-Share-Web](https://github.com/zkeyoung/Easy-Share-Web) .

## 安装

从 [GitHub Releases](https://github.com/zkeyoung/Easy-Share/releases) 下载安装。

## 把冰箱装进大象分几步 ?

1. 打开 Easy Share

2. 设置好你的配置

3. 选择你要分享的文件

4. 把下载链接发给你的兄弟
